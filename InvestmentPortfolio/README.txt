This program is designed to be a investment portfolio

To run the program

1. unzip the folder

2. load the project into netbeans

3. click the compile and run button


This program takes an investment class and puts them into a portfolio.
You can buy, sell, update, get gains, and search for your different investments.


Assumptions

Assumes that the text file will never differ from the format provided
Assumes that all values will always be populated in the text file
Assumes that the text files location is always in the Assets folder and it is called investments.txt

Test Plan
Users must enter full words, but is case insensitive 
All values must be entered when adding to the list
When searching at least one of the 3 values are required
